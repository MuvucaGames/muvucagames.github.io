Detailed Description

Input of shared information treatment.

Sensitive information which is shared between multiple instances must be treated with exclusive access.

Button for changing focus of keyboard and camera depends on who will be the next active controllable object. If next active is a Hero, we use button 'ChangeHero'. If next active is some other controllable object, for example, Claw, then user will change input and camera using the 'Action' button.

See also
Claw, ColorsPuzzle, Hero
